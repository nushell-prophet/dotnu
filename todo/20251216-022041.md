# Generate code from `#**` directive comments (the opposite of `dotnu embeds`)

## Idea

`dotnu embeds` runs commands and writes their **output** back into the file as `# =>` comments. I want the opposite: a command that runs a small pipeline written **inside a comment** and writes that pipeline's result back as **real code lines**, right after the comment.

The directive comment is never modified, so the generation can be re-run any time to refresh the code below it.

- Marker: `#**`. Everything after it on the line is a Nushell pipeline.
- The pipeline must return text. Each line of that text becomes one generated code line, inserted right after the directive.
- Re-running replaces the previously generated block, not the directive.

## Basic example

Source file:
```nushell
#** ls todo/ | get name | each { $"open -r ($in)" } | to text
```

After running the command:
```nushell
#** ls todo/ | get name | each { $"open -r ($in)" } | to text
open -r todo/20251216-022041.md
open -r todo/20260630-090000.md
```

Now the whole file is executable, and re-running the command refreshes the `open -r ...` lines whenever `todo/` changes.

## Why: integration tests triggered this

This repo's integration tests are the case that pushed me to write the idea down. Each test is a snapshot: run a command, save its output, compare against a fixture (`toolkit.nu`, `run-snapshot-test`). The list of tests is written by hand, so it can drift from the real fixtures, and a failure points at the whole block rather than one line.

A `#**` directive turns that list into generated code driven by the fixtures that actually exist on disk:
```nushell
#** ls tests/output-yaml/*.yaml | get name | path parse | get stem | each { $"    (run-snapshot-test '($in)' ...)" } | to text
```
The generated lines stay explicit — one call per line — so a failing test traces to a line number, and the list can never fall out of sync with the fixtures.

(The exact tests were reworked since I first had this idea; the point is the pattern, not these specific calls.)
